Confision Matrix

Condition Positive Condition Negative
Prediction Positive TP FP
Prediction Negative FN TN
  • TP = True Positive
  • TN = True Negative
  • FP = False Positive
  • FN = False Negative
  • N = TP + TN + FP + FN

Performance Metrics

  • Accuracy

\[ \text{acc} = \frac{TP + TN}{N} \]

  • Kappa

\[ \kappa = \frac{p_o - p_e}{1 - p_e} \]

where

  • \( p_e \) = Expected Accuracy (random chance).
  • \( p_o \) = Observed Accuracy.

The kappa metric can be thought as a modification of the accuracy metric based on the class proportions.

Performance Metrics

  • Sensitivity (= Recall)

\[ \text{sens} = \frac{TP}{TP + FN} \]

  • Specitivity

\[ \text{spec} = \frac{TN}{TN + FP} \]

  • Precision

\[ \text{prec} = \frac{TP}{TP + FP} \]

  • AUC

This metric refers to the area under the curve of the ROC curve. It does not depend on the cutoff probability threshold for the predicted classes.
